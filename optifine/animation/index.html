# 动态材质（后篇）

前置知识

​ 动态材质（前篇）

Minecraft原版只可以为物品和方块添加动态材质，OptiFine拓展了动态材质的功能，使动态材质可以用于几乎一切材质。

本章将展示若干实例，展示动态材质在方块和物品之外的各种应用。
